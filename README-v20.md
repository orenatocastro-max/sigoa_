# SIGOA v20 - Base SIGTAP única com importação TXT

Inclui:

- Base SIGTAP central e única.
- Importação de arquivo TXT pelo Administrador.
- Leitura de arquivos no padrão `COD. SIGTAP` e `DESC. SIGTAP`.
- Cadastro automático apenas dos códigos novos.
- Opção para manter ou atualizar descrições existentes.
- Histórico de importações com total lido, novos cadastrados, existentes, atualizados e erros.
- Arquivo `Cirurgias.txt` já incorporado à base inicial de SIGTAP.

## Como usar

Acesse como Administrador:

`Base SIGTAP -> Importar TXT`

Selecione o arquivo `.txt` e clique em **Importar base TXT**.

O sistema fará a validação pelo código SIGTAP. Se o código já existir, não será duplicado.
