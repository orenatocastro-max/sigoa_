# SIGOA v19 - Dashboard de subescalas sempre visível

Melhorias:
- Dashboard/Painel de Escalas mostra subescalas diretamente, sem clique.
- Pendentes aparecem primeiro, depois parciais e completas.
- Serviços sem subescala usam uma escala principal com nome amigável: Ressonância Magnética, Tomografia Computadorizada, Mamografia, Ultrassonografia etc.
- Oferta total por prestador/serviço aparece em cada card.
- Rol de procedimentos permanece separado das subescalas.
