# SIGOA v16 — Gestão modular de contratos, escalas e Rede Executora

Esta versão reorganiza o sistema em três micro-universos:

1. Gestão de Contratos: administrador cadastra prestadores, serviços e rol de procedimentos/subescalas executados.
2. Gestão de Escalas/Ofertas: operador lança apenas a escala/quantitativo da competência para os prestadores vinculados.
3. Rede Executora: consulta pública/gerencial da rede cadastrada.

## Novidades

- Dashboard de escalas com subescalas visíveis.
- Status automático por prestador + serviço:
  - ✅ Completa: todas as subescalas preenchidas.
  - ⚠️ Parcial: algumas subescalas preenchidas.
  - ❌ Pendente: nenhuma subescala preenchida.
- O operador não marca status manualmente; o sistema calcula pelo lançamento.
- Relatório de escalas por subescalas.
