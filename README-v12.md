# SIGOA Profissional v12

Alterações desta versão:

- Operador pode vincular/lançar procedimentos SIGTAP no módulo Oferta Mensal dos prestadores vinculados.
- Operador continua restrito apenas às unidades vinculadas ao seu cadastro.
- Telas e relatórios passam a exibir o tipo de serviço junto ao prestador sempre que possível.
- Administrador pode editar o cadastro do usuário, incluindo nome, login, perfil, status, motivo e vínculos com prestadores.
- Relatórios de prestadores passam a consolidar por prestador + tipo de serviço.

Após substituir os arquivos:

git add .
git commit -m "Adiciona procedimentos para operador e edicao de usuarios"
git push
